## 원리
촬영한 사진내에 있는 길이가 표준화 되어 있는 물체를 선택시, 그 물체를 기반으로 나머지 물체들의 대략적인 길이 정보를 반환.  
  
정면에서 찍은 사진이 아닌 경우 오차율이 큰 편이며, 깊이 카메라를 활용해 피사체와의 거리 정보를 알고 있다면 더 좋은 로직으로 발전이 가능할 것이다.
